Salep SNN: Advanced Neuromorphic CyberneticsSalep SNN is an open-source research platform dedicated to pioneering Spiking Neural Network (SNN) architectures that transcend the limitations of traditional von Neumann computing and current "point-neuron" deep learning paradigms.🧠 OverviewThis project represents a dual-phase research initiative in neuromorphic engineering:Phase I: Quadratic Learning Engine (Operational)We have implemented a custom SNN backend utilizing a Quadratic Spike-Rate Learning Rule. By moving beyond linear approximations, our engine achieves superior convergence rates and energy efficiency, making it ideal for edge-computing environments like autonomous greenhouse monitoring (Salep Sibernetiği).Phase II: Dendritic Computation & Phase Coding (Conceptual/Research)We are currently prototyping a next-generation architecture based on Multi-Compartmental Dendritic Neurons. Instead of relying on brute-force parameter scaling, this approach models individual neurons as independent "mini-cores" capable of:Dendritic Branching: Localized non-linear processing within the neuron.Phase-Coded Multiplexing: Encoding multiple data streams across sub-frequency bands, bypassing the physical interconnect bottlenecks ("Memory Wall").⚙️ Core Technical ArchitectureThe engine operates as an asynchronous, event-driven system:PlaintextFrontend (Dashboard) ↔ WebSocket (Real-time) ↔ Backend (Python SNN Engine)

📐 The Quadratic Learning ParadigmStandard SNNs often suffer from slow convergence and poor weight stability. Our approach derives plasticity from information-theoretic principles, maximizing mutual information between pre- and post-synaptic firing patterns:$$\Delta w = \eta \cdot \int_{t}^{t+dt} G(r_{pre}, r_{post}) \cdot Q(r_{pre}, r_{post}) \,dt$$Where $Q$ represents our proprietary quadratic interaction term that dynamically adjusts synaptic weights based on temporal coincidence detection.🔬 Future Roadmap: The Dendritic ShiftWhile Phase I is functional and actively deployed in our local cybernetic systems, Phase II is under intensive R&D. Our objective is to transition from traditional "Point Neurons" to SAM (Self-Adaptive Multicompartmental Neuron) Models. By simulating active ion channels within dendritic branches, we aim to increase the "internal expressivity" of a single neuron by a factor of 5–8, effectively turning a single node into a deep-learning processing unit.📂 Repository Structure/main.py: The asynchronous Python SNN engine./frontend.html: Real-time telemetry dashboard for neuro-metric visualization./docs/Proposal_Dendritic_Computing.pdf: Comprehensive research proposal on dendritic branching and frequency multiplexing.🚀 Getting StartedEnvironment: pip install -r requirements.txtBackend: uvicorn main:app --host 0.0.0.0 --port 8000Visualization: Open frontend.html in any modern web browser.📜 LicenseThis project is released under the MIT License.
